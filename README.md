# 💼 SmartTax SPK

> **เครื่องมือคำนวณภาษีเงินได้บุคคลธรรมดา (ภาษี SPK)** สำหรับประเทศไทยที่ออกแบบเพื่อความสะดวก รวดเร็ว และถูกต้อง

---

## 📋 เกี่ยวกับโครงการ

**SmartTax SPK** เป็นเว็บแอปพลิเคชันที่ช่วยให้คุณคำนวณภาษีเงินได้บุคคลธรรมดา (ภาษี SPK) อย่างง่ายและรวดเร็ว โดยอัปเดตผลลัพธ์แบบเรียลไทม์พร้อมกราฟวิเคราะห์โดยละเอียด ไม่จำเป็นต้องติดตั้งหรือมีพื้นฐานเทคนิค เพียงเปิดไฟล์และเริ่มคำนวณได้เลย

---

## ✨ ฟีเจอร์หลัก

| ฟีเจอร์ | รายละเอียด |
|--------|-----------|
| 🔢 **คำนวณแบบ Real-time** | ป้อนข้อมูลและรับผลลัพธ์ทันที ไม่ต้องรอ |
| 📊 **แสดงกราฟและสถิติ** | มองเห็นภาพรวมการจ่ายภาษีของคุณได้ชัดเจน |
| ⏱️ **เลือกช่วงเวลา** | คำนวณได้แบบต่อปี เดือน วัน หรือชั่วโมง |
| 💰 **ปรับค่าลดหย่อน** | รองรับค่าลดหย่อนด้านต่างๆ ตามกฎหมายภาษี |
| 🎨 **ดีไซน์สมัยใหม่** | อินเทอร์เฟซที่ใช้งานง่ายและรูปแบบสวยงาม |

## 🚀 วิธีใช้งาน

### 1️⃣ **ใช้งานออนไลน์ (แนะนำ)**
เยี่ยมชมเว็บไซต์โดยตรง:
```
https://jasimogfx.github.io/SmartTaxSPK.github.io/smarttax-spk.html
```

### 2️⃣ **ใช้งานเพื่อมพื้นฐาน**
1. ดาวน์โหลดไฟล์ `smarttax-spk.html`
2. เปิดไฟล์ด้วยเบราว์เซอร์ (Chrome, Firefox, Safari ฯลฯ)
3. เริ่มใช้งานได้ทันที ไม่ต้องติดตั้งอะไรเพิ่มเติม

### 3️⃣ **รันบนเซิร์ฟเวอร์ท้องถิ่น**
หากต้องการรันบนเซิร์ฟเวอร์โลคัล:

```bash
# ใช้ Python
python3 -m http.server 8000

# หรือใช้ Node.js
npx http-server
```

จากนั้นเปิด: `http://localhost:8000/smarttax-spk.html`

## 📖 วิธีคำนวณ

1. **ป้อนรายได้**: ใส่จำนวนรายได้ของคุณในปี/เดือน/วัน/ชั่วโมง
2. **ตั้งค่าลดหย่อน**: ปรับค่าลดหย่อนต่างๆ ตามที่ใช้บัญชี
3. **ดูผลลัพธ์**: ระบบจะคำนวณและแสดงจำนวนภาษีที่ต้องจ่ายทันที
4. **วิเคราะห์**: ศึกษากราฟและสถิติเพื่อเข้าใจการจ่ายภาษีของคุณ
